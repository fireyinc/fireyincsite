What is SpaceWorld 2

SpaceWorld 2 is a self run Minecraft 1.17.1 Server. It's original intent was to be a more modern counterpart for ActionMC, but as more people joined, it slowly turned into something COMPLETELY different. Now it's a server to build ANYTHING you want. Be it towns, static builds, homes, and more!

Enjoy!
